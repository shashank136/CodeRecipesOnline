What are data structures?

A data structutre is a way to store and organize data in order to facilitate access and modifications.No single data structure works well for all the purpose, and so it is important to know the strengths and limitations of several of them.


Types of Data Structures

Data structures can be categorized into two types: built-in data structures and user defined data structures. Built-in data structures are integer, float, character, pointers. Where as user defined data structures are : arrays,queues,linked list, trees, graphs.

Operations that can be performed on data structures are: insertion, deletion, searching.
